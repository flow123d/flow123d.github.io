Flow123d

Features

Flow123d is a solver for simulation of saturated water flow and transport of dissolved substances in fractured porous media. Principal language C++. Its key features are:

Complex Geometries Consisting of Elements of Different Dimensions

  • Convenient for modeling 2D and 1D fractures
  • Native support of GMSH meshes
  • Compatible interconnection of geometries
  • Non-compatible interconnections (under development)
  • Read more ...

Darcian Flow in Fractured Porous Media

  • Coupling on interfaces between adjacent dimensions
  • Mixed-hybrid formulation providing accurate velocity approximation
  • Efficient implementation using Schur complement
  • Unsteady model for non-zero storativity
  • Read more ...

Advection-Diffusion Equation for Solute Transport

  • Simple finite volumes with explicit timestepping for pure transport (without diffusion)
  • Robust Discontinuous Galerkin + implicit timestepping for transport with hydrodynamic diffusion
  • Support for arbitrary number of substances
  • Read more ...

Reactions, Adsorption, Dual Porosity

  • Adsorption and reaction models are coupled with transport models using the operator splitting method
  • Efficient implementation of linear and non-linear adsorption using interpolation tables
  • Decays
  • Dual porosity model

Parallel Computing

  • Parallel assembly, Schur complements, solver for Darcy flow model
  • Parallel finite volume explicit transport
  • Currently scalable up to 100 processors

Input and Output interface

  • Structured input using JSON file format
  • Almost all parameters can be arbitrary space-time field
  • Output to VTK and GMSH formats

Active Development

  • XFEM method for modelling wells (1D-3D coupling)
  • Methods for coupling of equations (Darcy flow) on non-compatible 2D-3D and 1D-2D meshes modeling preferential paths. Paper
