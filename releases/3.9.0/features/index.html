Features

Flow123d is a simulator of mechanical and transport processes in fractured porous media. The simulator consistently use the continuum-fracture approach combining classical continuum equations with similar equations on the discrete fractures represented as 2d manifolds. The physically based equations on the fractures and coupling with the continuum are derived by the integration accross the fracture apperture and suitable semi-approximation. The resulting method allows continuous transition between continuum and Discrete Fracture Network description of the fractured rock.

Support for complex geometries

  • Convenient for modeling 2d fractures and 1d channels in 3d ambient media
  • The BGEM tool is suited for prepartion of the meshes including the stochastic fracture networks. It essentialy use GMSH meshing software.

Darcian Flow in Fractured Media

  • Saturated and variably saturated (Richards) model.
  • Coupling on interfaces between adjacent dimensions
  • Mixed-hybrid formulation providing accurate velocity approximation
  • Efficient implementation using Schur complement
  • Read more ...

Advection-Diffusion Equation for Solute Transport

  • Finite Volumes model for pure advection. Explicit scheme.
  • Robust Discontinuous Galerkin + implicit scheme for transport with hydrodynamic diffusion
  • Support for arbitrary number of substances
  • Read more ...

Reaction term

  • Reaction term includes: dual porosity model, sorptions, and linear reactions (decays)
  • Coupled with transport models using the operator splitting method
  • Support for general combination of several reaction processes.
  • Efficient computation of non-linear sorption via. interpolation
  • Read more ...

Heat transfer

  • Equilibrium heat transfer model including conduction in rock and conduction and convection in liquid phase.
  • Robust Discontinuous Galerkin + implicit scheme.

Mechanics

  • Linear elasticity.
  • Coupled with Darcy flow. Biot Poroelasticity.
  • Contacts on fractures, non-linear dependency of permeability.
