Features

Flow123d is a solver for simulation of saturated and unsaturated water flow and transport of dissolved substances in fractured porous media. Principal language C++. Main features include:

Complex Geometries Consisting of Elements of Different Dimensions

  • Convenient for modeling 2d fractures and 1d channels in 3d ambient media
  • Native support of GMSH meshes
  • Compatible interconnection of geometries
  • Non-compatible interconnections (under development)
  • Read more ...

Darcian Flow in Fractured Porous Media

  • Both saturated and variably saturated cases.
  • Coupling on interfaces between adjacent dimensions
  • Mixed-hybrid formulation providing accurate velocity approximation
  • Efficient implementation using Schur complement
  • Unsteady model for non-zero storativity
  • Read more ...

Advection-Diffusion Equation for Solute Transport

  • Simple finite volumes with explicit timestepping for pure transport (without diffusion)
  • Robust Discontinuous Galerkin + implicit timestepping for transport with hydrodynamic diffusion
  • Support for arbitrary number of substances
  • Read more ...

Reaction processes

  • Reaction processes include: dual porosity model, sorptions, and linear reactions (decays)
  • Coupled with transport models using the operator splitting method
  • Support for general combination of several reaction processes.
  • Efficient computation of non-linear sorption via. interpolation

Heat transfer

  • Equilibrium heat transfer model including conduction in rock and conduction and convection in liquid phase.
  • Robust Discontinuous Galerkin + implicit timestepping.

Software specific

  • Parallel assembly, solvers, calculation of reaction term.
  • BDDC solver, paper.
  • Unified balance output of conserved quantities.
  • Structured input using YAML file format
  • Almost all parameters can be arbitrary space-time field
  • Output to VTK and GMSH formats

Active Development

  • XFEM method for modelling wells (1D-3D coupling)
  • Methods for coupling of equations (Darcy flow) on non-compatible 2D-3D and 1D-2D meshes modeling preferential paths. Paper
